FILE-SYSTEM-ORGANISER

Runs globally in a directory the user wishes(use 'edith' as the global command)

The Above global command can also be changed to anything you want, Follow the below steps to change it
    1. Go to the package.json file 
    2. Navigate to bin and change the key for file "main.js"

## List of commands

1. edith tree => Produces a tree structure of the files in current directory

2. edith organize => Ogranises all the files on the basis of videofile, audiofile , textfile etc. in the current directory into a folder in that directory itself.

3. edith help => produces the list of all the commands available on terminal
